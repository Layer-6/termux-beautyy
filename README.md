# 🎨 Features

# · Custom MOTD (Banner): A unique ASCII logo + your own text (e.g., Telegram channel) – pick any color for both.
# · Custom Hacker Prompt: The classic ┌─[💀 user@host]─[~] style with full customization.
# · Personalize Everything:
 # · Emoji of your choice (💀, 👻, 🔥, 🚀, …)
#  · Username and hostname
#  · Prompt color (Red, Green, Yellow, Blue, Purple, Cyan, White)
# ·Safe & Reliable: Automatically backs up your original settings. You can always reset to default.
# ·Simple Menu – no complex flags needed.

# 🖌️ Prompt Color Guide

# When customizing the prompt, you can choose from 7 different colors:

# Number Color Code Hex
# 1 Red \033[1;31m #FF0000
# 2 Green \033[1;32m #00FF00
# 3 Yellow \033[1;33m #FFFF00
# 4 Blue \033[1;34m #0000FF
# 5 Purple \033[1;35m #FF00FF
# 6 Cyan \033[1;36m #00FFFF
# 7 White \033[1;37m #FFFFFF

# The banner logo and your extra text can also be colored independently using a smaller palette (Black, Gray, Red, Green, Purple for the logo; plus Cyan and White for text).
